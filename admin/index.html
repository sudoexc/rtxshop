Конфигурация Telegram бота и системы

Telegram Chat ID

Укажите Chat ID группы или канала, куда будут приходить заявки. Чтобы узнать ID — добавьте бота в чат и нажмите «Получить обновления».

Тест соединения

Отправьте тестовое сообщение в указанный чат, чтобы убедиться что всё настроено верно.

Получить Chat ID автоматически

Напишите любое сообщение боту (или в группу с ботом), затем нажмите «Получить обновления». Нажмите на ID чата чтобы сохранить его автоматически.
